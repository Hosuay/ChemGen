## ✨ Key Features

### 🧬 **1. SMILES Input & Processing** (Module 01)
- **5-stage preprocessing pipeline**:
  1. Validation (chemical correctness)
  2. Cleaning (standardization)
  3. Canonicalization (unique representation)
  4. Error correction (heuristic fixes)
  5. Feature extraction (initial descriptors)
- InChI-based duplicate detection
- PAINS & reactive group filtering
- Stereochemistry handling
- Batch processing support

### 🔬 **2. Molecular Variant Generation** (Module 02)
- **27-block latent space partitioning** (3³ harmonic)
- Similarity-guided mutations (Tanimoto ≥ threshold)
- Scaffold preservation (Murcko scaffolds)
- Bioisosteric replacements
- Fingerprint-based filtering (Morgan ECFP)
- **3-6-9 iterative refinement cycles**
- 36 variants per input molecule (6² hexadic squared)

### 📊 **3. Molecular Property Analysis** (Module 03)
- **33 primary molecular descriptors**:
  - **Physicochemical** (9): MolWt, LogP, MR, HBD, HBA, TPSA, etc.
  - **Structural** (12): Rings, chirality, complexity, etc.
  - **Topological** (6): Connectivity indices, shape metrics
  - **Quantum** (6): Electronic properties, reactivity
- **9-step post-processing**:
  1. Normalization
  2. Outlier detection
  3. Missing value imputation
  4. Feature scaling
  5. Correlation analysis
  6. Dimensionality assessment
  7. Harmonic labeling
  8. Quality validation
  9. Export preparation
- Lipinski's Rule of Five compliance
- QED (Quantitative Estimate of Drug-likeness)

### 🤖 **4. ML-Assisted Evaluation** (Module 04)
- **7-layer deep neural networks**
- **3-6-9 training cycle pattern**:
  - 3 warmup cycles
  - 6 main training cycles
  - 9 fine-tuning cycles
- Real models trained on ESOL dataset (solubility)
- Ensemble learning for robust predictions
- Uncertainty quantification (standard deviation)
- Cross-validation (5-fold)
- Predictions:
  - Solubility (log mol/L)
  - Toxicity (synthetic)
  - Synthetic feasibility
- **Random seeds aligned to multiples of 27**

### 🌐 **5. 3D Visualization** (Module 05)

## 🛠️ Installation

### Prerequisites

- Python 3.8+ (primary language)
- Optional: C++17, Rust 1.60+, Julia 1.7+ (for performance modules)
- 12GB RAM recommended
- GPU optional (accelerates ML training)

### Quick Install (Python)

```bash
# Clone repository
git clone https://github.com/Hosuay/VariantProject.git
cd VariantProject

# Install dependencies
pip install -r requirements.txt

# For RDKit (chemistry toolkit)
conda install -c conda-forge rdkit

# Verify installation
python config/harmonic_config.py
```

### Full Installation (Multi-language)

```bash
# Install Python dependencies
pip install -r requirements.txt

# Install C++ dependencies (optional, for performance)
cd modules/03_molecular_analysis/cpp
mkdir build && cd build
cmake ..
make

# Install Rust bindings (optional)
cd modules/02_variant_generation/rust
cargo build --release

# Install Julia packages (optional)
julia modules/04_ml_scoring/julia/setup.jl
```

## 🚀 Quick Start

### Basic Usage (Python)

```python
from modules.harmonic_explorer import HarmonicVariantExplorer

# Initialize explorer
explorer = HarmonicVariantExplorer()

# Define input molecules
smiles_list = [
    'CC(=O)Oc1ccccc1C(=O)O',       # Aspirin
    'CN1C=NC2=C1C(=O)N(C(=O)N2C)C' # Caffeine
]

# Run exploration with harmonic parameters
results = explorer.explore(
    smiles_list=smiles_list,
    n_variants=36,  # 6² hexadic squared
    latent_blocks=27,
    use_sacred_geometry=True
)

# View top 9 molecules (enneadic display)
print(results.head(9))

# Visualize Pareto frontier
explorer.visualize_pareto_frontier()

# Export harmonized results
explorer.export_harmonic_csv()
```

### Advanced Usage with All Features

```python
explorer = HarmonicVariantExplorer(
    enable_pains_filter=True,
    enable_27_block_partition=True,
    ml_layers=7,
    training_cycles='3-6-9',
    post_processing_stages=9
)

results = explorer.explore_with_full_pipeline(
    smiles_list=smiles_list,
    preserve_scaffold=True,
    use_bioisosteric=True,
    compute_33_descriptors=True,
    apply_pareto_optimization=True,
    generate_3d_structures=True,
    harmonic_visualization=True
)
```
